今生一场不断向上的修行

发表于 2999-06-28 | 分类于 个人感悟

老先生

  前段时间，和同学去隔壁大学听一位老教授义务讲的《易经》课，老教授今年90多岁，温文尔雅，退休后一直坚持义务讲授传统国学。上课后，在老教授的学生中间我意外的发现有10几位和他年纪相仿的老人都在认真的记着笔记，认真的程度远超我们这帮毛头小子。他们所表现出来的对自我成长的追求，严重地冲击着我以往对学习目的的认知。

  其实人生就是一场不断向上的修行，我们不断的提升着自己，期望和更好的自己在明天不期而遇，期望活成自己想要的样子。我们慢慢的发现，无论到了什么年龄，自我成长都是必不可少的功课。

自我成长包含思想的提升和专业技能的提高两个方面。

  当我无牵无挂的时候，贫穷对于我来说只是晚上吃馒头和牛排的区别，无损我的快乐，可是当我爱上一个姑娘的时候，我才深深的感受到贫穷所带来的自卑。

  专业技能的提高，能够让我们在不断地自我成长中获得外在的认可，得到报酬，从而为自己和家人提供生活的物质基础。我们不断地追求自我成长，并不意味着对外在成功的鄙视，它也是自我成长的一部分。试想一下，如果有一天我们遇到了想要照顾一生的人却发现承担不起，这将是何等的痛苦。

  但外在的成功从来都不是自我成长的归宿，最后的归宿一定指向我们自己，人不同与野兽，野兽都是凭借本能去生存，而人喜欢思考，我们对我们存在的星球，对人类的历史充满着好奇，我们会思考和探索人生的意义，想要找到自己在这个社会中扮演的角色。

  当我们找不到人生目的在那里时，找不到自己的定位时，就会产生空虚和迷茫感，试着回忆一下你大学刚开始的时光，是否也像大多数人那样产生了迷茫感。面对这种迷茫感，有一小部分的人会通过阅读与智者沟通，向内询问自己想要什么样的生活；通过行动，与社会发生联系，向外寻找自己兴趣与社会的交集点。在这个交集点不断提高自己的专业技能，找到自己的定位。但也有人因为各种原因，选择随波逐流，选择飘飘荡荡，终其一生活在茫然里，找不到自己的归宿。

发表于 2017-10-25 | 分类于 javaScript学习总结

javaScript这个迷人又令人困惑的混血儿，借鉴了多种编程语言的机制，这其中就包括就包括this机制。

this是什么？

在javascript里面，this是一个特殊的对象，它不像其他编程语言那样，是存储在实例中的值，直接指向此实例。而是作为一个单独的指针，自动定义于每一个函数域中，指向不同的位置，这也是为什么我们会将它搞混的原因。

为什么使用this

this为代码复用提供了可能，让代码变得更加简洁和复用。

this的5条规则

this到底绑定或者引用那个对象环境决定于函数被调用的地方。而函数的调用有着不同的方式，在不同的方式中调用决定this引用的是那个对象。

->我们一般只研究函数执行的时候里面的this

->this是谁和当前的函数在哪执行和在哪定义没有半毛钱的关系

1)看函数执行的时候,函数名之前是否有”.”,有的话”.”前面是谁this就是谁,没有的话this就是window

oDiv.onclick=function(){

//匿名函数中的this->oDiv

2)自执行函数中的this永远是window

3)给元素的某一个事件绑定方法,当事件触发的时候,方法中的this是当前元素本身

4)在构造函数模式中,我们的this.xxx=xxx中的this是当前的类的一个实例

function Fn(){

this.x=100;//this->f x是给当前实例f增加的私有的属性

}

console.log(this.x);

};

var f=new Fn;

f.getX();//getX中的this->f

f.proto.getX();//getX中的this->Fn.prototype

5)call和apply强制改变this的指向->以上所有的this情况在遇到call/apply的时候都不好使,都已强制改变的为主

一般情况下,我们执行call方法第一个传递的参数值是谁,那么fn中的this就是谁
