今生一场不断向上的修行

发表于 2999-06-28 | 分类于 个人感悟

老先生


  前段时间，和同学去隔壁大学听一位老教授义务讲的《易经》课，老教授今年90多岁，温文尔雅，退休后一直坚持义务讲授传统国学。上课后，在老教授的学生中间我意外的发现有10几位和他年纪相仿的老人都在认真的记着笔记，认真的程度远超我们这帮毛头小子。他们所表现出来的对自我成长的追求，严重地冲击着我以往对学习目的的认知。

  其实人生就是一场不断向上的修行，我们不断的提升着自己，期望和更好的自己在明天不期而遇，期望活成自己想要的样子。我们慢慢的发现，无论到了什么年龄，自我成长都是必不可少的功课。

—

自我成长包含思想的提升和专业技能的提高两个方面。

  当我无牵无挂的时候，贫穷对于我来说只是晚上吃馒头和牛排的区别，无损我的快乐，可是当我爱上一个姑娘的时候，我才深深的感受到贫穷所带来的自卑。

  专业技能的提高，能够让我们在不断地自我成长中获得外在的认可，得到报酬，从而为自己和家人提供生活的物质基础。我们不断地追求自我成长，并不意味着对外在成功的鄙视，它也是自我成长的一部分。试想一下，如果有一天我们遇到了想要照顾一生的人却发现承担不起，这将是何等的痛苦。

  但外在的成功从来都不是自我成长的归宿，最后的归宿一定指向我们自己，人不同与野兽，野兽都是凭借本能去生存，而人喜欢思考，我们对我们存在的星球，对人类的历史充满着好奇，我们会思考和探索人生的意义，想要找到自己在这个社会中扮演的角色。

  当我们找不到人生目的在那里时，找不到自己的定位时，就会产生空虚和迷茫感，试着回忆一下你大学刚开始的时光，是否也像大多数人那样产生了迷茫感。面对这种迷茫感，有一小部分的人会通过阅读与智者沟通，向内询问自己想要什么样的生活；通过行动，与社会发生联系，向外寻找自己兴趣与社会的交集点。在这个交集点不断提高自己的专业技能，找到自己的定位。但也有人因为各种原因，选择随波逐流，选择飘飘荡荡，终其一生活在茫然里，找不到自己的归宿。

python安装及环境搭建

本文介绍如何在本地安装及搭建Python开发环境。

Python是跨平台的编程语言，它可以运行在Windows、Mac和各种Linux/Unix系统上。在Windows上写Python程序，放到Linux上也是能够运行的。要开始学习Python编程，首先就得把Python安装到你的电脑上。安装后，你会得到Python解释器（就是负责运行Python程序的），一个命令行交互环境，还有一个简单的集成开发环境。

你可以通过终端窗口输入 “python” 命令来查看本地是否已经安装Python以及Python的安装版本。

Python官网


Python最新源码，二进制文档，新闻资讯等可以在Python的官网查看到：
Python官网：http://www.python.org/
你可以在以下链接中下载 Python 的文档，你可以下载 HTML、PDF 和 PostScript 等格式的文档。
Python文档下载地址：www.python.org/doc/

Python下载、安装

以下为不同平台上安装Python的方法：

Unix & Linux 平台安装 Python:

  • 以下为在Unix & Linux 平台上安装 Python 的简单步骤：
  • 打开WEB浏览器访问http://www.python.org/download/
  • 选择适用于Unix/Linux的源码压缩包。
  • 下载及解压压缩包。
  • 如果你需要自定义一些选项修改Modules/Setup
  • 执行 ./configure 脚本
  • make
  • make install

执行以上操作后，Python会安装在 /usr/local/bin 目录中，Python库安装在/usr/local/lib/pythonXX，XX为你使用的Python的版本号。

Window 平台安装 Python:


以下为在 Window 平台上安装 Python 的简单步骤：

  • 打开WEB浏览器访问http://www.python.org/download/
  • 在下载列表中选择Window平台安装包，包格式为：python-XYZ.msi 文件 ， XYZ 为你要安装的版本号。
  • 下载后，双击下载包，进入Python安装向导，安装非常简单，你只需要使用默认的设置一直点击”下一步”直到安装完成即可。
